# Git and Markdown Basics

## What is Git?
Git is a version control system used to track changes in code. It helps developers manage versions of a project and collaborate with others.

## How to Push Code to GitHub
1. Initialize repository
git init

2. Add files
git add .

3. Commit changes
git commit -m "first commit"

4. Connect to GitHub
git remote add origin <repo-url>

5. Push code
git push origin main

## What is Markdown?

Markdown is a simple language used to format text in files like README.md.

## Basic Markdown Syntax

### Headings
# Heading 1
## Heading 2

### Lists
- Item 1
- Item 2

### Code Block
  print("Hello world")

### Links
[GitHub](https://github.com)
